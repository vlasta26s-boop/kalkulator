## Soubory

Každý formulář je **jediný samostatný HTML soubor** – obrázky jsou vloženy přímo do souboru (base64). Není potřeba žádný další soubor.

| Soubor | Formulář | Velikost |
|---|---|---|
| `autopojisteni.html` | 🚗 Autopojištění | ~480 KB |
| `urazove_pojisteni.html` | 🩺 Úrazové pojištění | ~1,5 MB |
| `pojisteni_majetku.html` | 🏠 Pojištění majetku | ~490 KB |
| `index.html` | Rozcestník | 4 KB |

## Použití

### Samostatně (bez serveru)
Stačí otevřít soubor v prohlížeči – funguje i z lokálního disku.

### Přes GitHub Pages
1. Nahrajte soubory do GitHub repozitáře
2. Settings → Pages → Source: `main / root`
3. Formuláře budou dostupné na:

```
https://<váš-účet>.github.io/<repo>/autopojisteni.html
https://<váš-účet>.github.io/<repo>/urazove_pojisteni.html
https://<váš-účet>.github.io/<repo>/pojisteni_majetku.html
```
